# Courses-Project
School Courses Management System
Overview
The School Courses Management System is a comprehensive web platform designed to automate and manage online learning and training processes, from student registration to content delivery, payment handling, and efficient reporting. This system is designed to connect Students, Instructors, and Admins in an integrated and organized environment.
The main goal is to replace manual processing, which causes confusion, delays, and human error, with an automated system that improves accessibility, organization, and transparency.
 Key Functional Requirements
The platform supports three user categories, each with a specific set of functionalities:

Students
Register and log in to the system.
Browse available courses and submit applications.
Track application status (On Hold $\rightarrow$ Accepted $\rightarrow$ Payment $\rightarrow$ Active).
Complete payment and unlock access to course materials.
Access course materials, schedules, and Zoom links.
Submit assignments/tasks and receive notifications about deadlines or updates.
Download digital certificates upon course completion.


Instructors
Upload and manage educational materials (PDFs, videos, etc.).
Manage course sessions and add Zoom links for live broadcasts.
Track student attendance and submissions.
Evaluate assignments and provide feedback and comments to students.
Send announcements and updates to all enrolled students.


Admins
Add, edit, and delete courses.
Approve or reject student applications.
Assign instructors to various courses.
Manage and verify payments.
Generate analytical reports and statistics.
Manage user roles and permissions within the system.



Non-Functional Requirements
Performance: Each page should load in under 3 seconds.
Usability: The interface must be responsive and intuitive on both desktop and mobile devices.
Security: All user passwords and payments must be encrypted using SSL.
Availability: System uptime should not fall below $99 \
Localization: Support for a multilingual interface (Arabic and English).
